本软件是一款基于深度学习的人脸识别系统，结合了Haar特征进行人脸检测。界面设计简洁明了，功能如下：

功能概述

  1. 实时人脸检测与识别：

    • 左侧显示摄像头捕捉的实时图像。
    • 右侧显示识别出的人脸图像。

  2. 人脸添加：

    • 用户可以通过点击“Add”按钮添加新的面部数据。
    • 输入框允许用户为新增的人脸数据命名。

  3. 识别信息展示：

    • 实时显示识别到的人脸名称。
    • 显示识别相似度分数，方便用户了解识别准确性。

  4. 日志记录：

    • 底部记录详细的识别信息和日志，包括时间、相似度、人脸名称等。
    • 方便用户追踪识别记录和系统状态。

技术细节

  • 人脸检测：

    • 使用Haar特征分类器快速检测图像中的人脸区域。

  • 人脸识别：

    • 基于深度学习算法（如卷积神经网络，CNN）对检测到的人脸进行特征提取和比对。
    • 高效的识别算法确保识别的准确性和实时性。

使用说明

  1. 启动软件后，点击“Start”按钮开始人脸识别。
  2. 将人脸对准摄像头，系统会自动检测并显示在左侧窗口。
  3. 添加新面部数据：
    • 点击“Add”按钮。
    • 在右侧输入框中输入名称，然后系统会记录该人脸数据。
  4. 系统自动在右侧显示识别出的人脸，并在下方日志记录详细信息。
